Hola, Bienvenido!

Pagina desarrollada para temas de pruebas, a continuación la descripción de los módulos que podrá encontrar:

  • Cliente: este módulo está creado para la administración de clientes y donde también se puede redireccionar al módulo de compras
  • Producto: ste módulo está creado para la administración de productos unicamente
  • Ventas: solo se puede acceder desde el módulo de cliente

Para iniciar es necesario angular y el servicio REST:

  • Configuración del proxy para CORS. desde el archivo de enviroment, solo cambien la ruta.
  • Otras configuraciones. Por el momento en ajustes.
  • Tecnologias adicionales. Bootstrap-angular y Notiflix.
